# Quick Setup Guide

## 1. Start GraphQL Server

```bash
npm install
npm run server
```

Keep this running in Terminal 1.

## 2. Choose Your Framework


cd leaderboard-app
ng serve
angular - readme contains set up information


## 3. Test GraphQL Server

Open browser: http://localhost:3000/graphql

## 4. Build Your App

- Leaderboard page with player rankings
- Player profile page with details
- Use GraphQL for all data fetching
- Make it responsive

Good luck!
